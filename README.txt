Défi - Morpion - 

CONSIGNES :

1/ Réalisez en HTML/CSS et JS le jeu du morpion.
2/ Prévoyez :
- Une zone pour afficher le nom du joueur qui doit positionner
sa marque (Rond ou Croix)
- Un bouton “start” pour commencer la partie
- Un bouton “restart” pour remettre à zéro la partie
3/ Pour l’aspect visuel, vous êtes libre !
4/ Vous pouvez utiliser le framework de votre choix.
5/ Les 3 meilleurs réalisations seront partagées sur le mur.
